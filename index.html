End-to-End RAG Pipeline with Ollama

This project implements a sophisticated Retrieval-Augmented Generation (RAG) pipeline using Ollama, Triplex, and LangChain. It processes PDF documents, extracts key information using Triplex, creates embeddings, and sets up a question-answering system. The pipeline includes a Gradio interface, allowing users to interact with the system and ask questions about the processed document. This showcase demonstrates advanced NLP techniques, including entity extraction, semantic chunking, and vector-based retrieval.
